Network of Financial Institutions

The circle size is proportional to the centrality of the node, measured via the debt rank. Hover over the circles to see more information about each bank. Use the widgests as follows:

Data: Select the netwoork representation to be used for the analysis

Stress Node: Select a node by clicking on it and then use this button to apply a given stress to the node and simulate its propagation throug the network. Clik on the Refresh icon to display the results and over over the circles to see the the impact on each node. The color intensity is higher for higher impact.

Stress Parameter: Define the level of stress to be applied to a given node. It needs to be a float between 0 (no stress) and 1 (default).

Back to Original: Go back to the original graph.
